## setup

1. install node
2. yarn init -y :setup project
3. yarn add typescript :install typescript
4. tsc --init :setup typescript
##### (NOTE: CHANGE NODE VERSION if tsc doesn't work)
5. tsc :compile to js  
6. node file.js : run the js compiled file
